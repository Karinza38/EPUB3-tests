AV Content Test - EPUB3FXL

Purpose

This test is to measure the level of support for AV Content in EPUB3FXL files.

The format is a simple pass/fail, if the content plays, the test passes, if not, the test fails.

For any test that fails, a defect should be filed against the device/app where the failure appeared. Testers should remember to check for existing defects before creating new ones.
